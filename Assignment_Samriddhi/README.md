# Student Directory React App

This is a polished React student directory designed to handle academic tracking through a highly interactive interface. It features comprehensive CRUD operations—enabling you to add, modify, and remove student records—alongside advanced search, filter, and sorting capabilities. The application is built for versatility, offering dark mode, a toggle between grid and list layouts, and localStorage integration to ensure your data persists across sessions.
---
 ##  🚀 Core Features Overview:

- Dynamic Data Management: Uses useState to manage the student list and handle real-time updates for status toggles or deletions. Like marking a student as Present or Absent, the application dynamically updates the list without requiring a page refresh.
- Advanced Searching: Utilizes the useState hook to maintain a search query, allowing users to filter students by name, course, or grade.
- Sorting and Filtering: Implements sorting options for students by name or grade, and filters by course and attendance status.  
- Adaptive UI: Employs plain CSS to provide a responsive experience that shifts between card-based grids and organized lists.
- Dark Mode: Implements a toggle for a user-friendly dark mode option.
- Conditional Intelligence: Features built-in logic to highlight Top Performers, display Present/Absent badges, and manage empty search      states.
